# PaveSTITCH-SynKpacK
Stitches pavement images from the sensor fusion of accelerometers, RGB-D camera, GPS and LED. Also, synchronizes the data from these sensors.

# Sensors required
Single or multiple Kinects <br />
Single or multiple accelerometers <br />
GPS <br />
LED <br />

# Requirements
MATLAB <br />
MATLAB - Image Processing Toolbox <br />
MATLAB - Signal Processing Toolbox <br />
